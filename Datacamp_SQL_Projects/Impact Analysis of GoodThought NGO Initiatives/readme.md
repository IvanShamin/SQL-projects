# GoodThought NGO Impact Analysis

SQL-based analysis of GoodThought NGO assignments, donations, and regional impact (2010–2023).

## Project Goals
- Identify top-impact assignment per region
- Analyze donation patterns by donor type and assignment

## Technologies
- PostgreSQL
- SQL (CTEs, window functions, JOINs, aggregation)
- Data visualization (in notebook)

## Key Queries
1. Top impact assignment per region (using ROW_NUMBER())
2. Assignments with highest rounded donation amounts
