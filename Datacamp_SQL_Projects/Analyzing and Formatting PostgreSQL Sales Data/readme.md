# Analyzing and Formatting PostgreSQL Sales Data

DataCamp project (completed February 2026)

## Project Goal
Clean and analyze a Super Store sales dataset:
- Identify top 5 products per category by sales/profit
- Impute missing `quantity` values using unit price calculated from discount/market/region

## Key Skills Demonstrated
- PostgreSQL queries
- CTEs (Common Table Expressions)
- Joins (USING)
- Aggregations (SUM, AVG)
- Window functions (RANK)
- Data cleaning & imputation (handling NULLs)
