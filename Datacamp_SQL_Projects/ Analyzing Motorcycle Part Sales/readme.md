# Motorcycle Parts Wholesale Revenue Analysis

**SQL project** — Net revenue calculation for wholesale orders only  
June–August 2021 | DataCamp / DataLab | February 2026

## Project Description

A company sells motorcycle parts through three warehouses: **North**, **Central**, **West**.  
They accept orders via retail and wholesale channels and support three payment methods (Credit card, Cash, Transfer), each with a different fee percentage.

**Business request**:  
Calculate **net revenue** (after payment fees) from **wholesale orders only**, grouped by:

- product_line  
- month (June, July, August)  
- warehouse
